## Selected projects

### ◆ SME Automation Portal — multi-tenant SaaS
A platform where small businesses can spin up and manage their own AI chatbot
services across WhatsApp, Facebook, and Instagram. Tenant isolation via row-level
security, tiered BDT pricing, and local payment rails (bKash · Nagad · Rocket).

`Next.js 14` · `Supabase` · `Vercel` · `RLS multi-tenancy`
→ [Repository](https://github.com/[your-handle]/sme-automation-portal)

### ◆ WhatsApp AI Ordering System
An end-to-end ordering agent that takes orders in natural language over WhatsApp,
confirms them, and writes structured records to a live operations sheet — no app,
no friction for the customer.

`Make.com` · `GPT-4o-mini` · `WhatsApp Cloud API` · `Google Sheets`
→ [Repository](https://github.com/[your-handle]/whatsapp-ordering-agent)

### ◆ Multi-Channel Support Agent
A support agent grounded in a business's own knowledge base that answers
questions consistently across channels and escalates cleanly when it can't help.

`OpenAI` · `RAG` · `Supabase (pgvector)` · `Webhooks`
→ [Repository](https://github.com/[your-handle]/support-agent)

### ◆ CineVault — JavaFX desktop application
An IMDb-style movie management system built as an OOP project. Java 21, a clean
four-layer architecture (model · persistence · service · UI), role-based access
control, and serialization-based storage. Demonstrates all four OOP principles
in a real, runnable app.

`Java 21` · `JavaFX` · `Layered architecture` · `OOP`
→ [Repository](https://github.com/[your-handle]/cinevault)
